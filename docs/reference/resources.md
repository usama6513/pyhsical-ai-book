## Getting Help

### When You're Stuck

1. **Search first** - Google your error message
2. **Check documentation** - Official docs usually have answers
3. **Ask on Stack Overflow** - Include minimal reproducible example
4. **Try Rubber Duck Debugging** - Explain your code to a rubber duck (works surprisingly well!)
