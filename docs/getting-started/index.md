# Getting Started

Welcome! This guide will help you set up and start learning from the Physical AI Book.

## 5-Minute Setup

Get the book running in just 5 minutes:

```bash
# 1. Clone or download the repository
git clone <repo-url>
cd physical-ai-book

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev

# 4. Open your browser
# The book opens automatically at http://localhost:3000
```

## Prerequisites

You'll need:
- **Node.js 18+** - [Download here](https://nodejs.org/)
- **npm** - Comes with Node.js
- **Basic programming knowledge** - Comfort reading and writing simple functions
- **Code editor** - Visual Studio Code, Sublime, or your favorite editor
