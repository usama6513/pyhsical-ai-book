# Welcome to Physical AI Book

Learn AI through hands-on examples and interactive lessons.

## What You'll Learn

This book is designed for beginners and intermediate learners who want to understand AI through practical, runnable code examples rather than just theory.

- **Hands-on Examples**: Every concept includes working code you can run immediately
- **Progressive Complexity**: Start with fundamentals, build to practical applications
- **Cross-Platform**: All examples work on Windows, macOS, and Linux
- **Self-Assessment**: Check your understanding at the end of each lesson
