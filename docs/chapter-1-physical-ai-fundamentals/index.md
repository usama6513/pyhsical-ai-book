# Chapter 1: Physical AI Fundamentals

Welcome to Chapter 1! In this chapter, you'll learn the foundations of Physical AI through hands-on examples and interactive lessons.

## What You'll Learn

- **Lesson 1**: What is Physical AI? - Core concepts and your first working example
- **Lesson 2**: Core Concepts & Intuition - Build deeper understanding with intermediate examples
- **Lesson 3**: Building Your AI Intuition - Practical applications and advanced patterns

## Time Estimate

- **Total Chapter Time**: ~90 minutes
- **Lesson 1**: 25 minutes
- **Lesson 2**: 30 minutes
- **Lesson 3**: 35 minutes

## Prerequisites

- Basic programming knowledge (you can read and write simple functions)
- A text editor or IDE
- Node.js 18+ installed (get it from [nodejs.org](https://nodejs.org/))

## Learning Path

### Recommended: Sequential Learning

Start with Lesson 1 and progress through Lesson 3. Each lesson builds on concepts from previous lessons:

1. [Lesson 1: What is Physical AI?](./lesson-1-fundamentals.md)
2. [Lesson 2: Core Concepts & Intuition](./lesson-2-core-concepts.md)
3. [Lesson 3: Building Your AI Intuition](./lesson-3-building-intuition.md)

### Optional: Topic-Based Learning

If you're jumping in:
- **Just want the basics?** Start with Lesson 1
- **Already familiar with fundamentals?** Try Lesson 2
- **Want practical applications?** Go to Lesson 3

## Structure

Each lesson follows this format:

1. **Learning Objectives** - What you'll understand
2. **Concept Overview** - Plain-language explanation
3. **Visual Diagrams** - See the concepts illustrated
4. **Working Examples** - 3 complete, runnable examples
5. **Try This Modifications** - Experiments to deepen understanding
6. **Key Takeaways** - Summary of main points
7. **Checkpoint** - Self-assessment questions
